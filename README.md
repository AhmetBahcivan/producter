<h1> Graphql Demo Basketbal Team Api </h1>
This is an demo basketball GraphQL API in Java.

Complete stack:

<li>Java</li>
<li>Spring Boot</li>
<li>GraphQL</li>
<li>Postgres</li>

Required to set up Postgres credentials and db name in application.properties
